# ✦ planner financeiro money ✦

Um planner financeiro web delicado, intuitivo e visual 💖 feito para organizar sua vida financeira de forma simples, prática e bonita **(pensado exclusivamente para uso no celular)**.

---

## ✨ Sobre o projeto

O **money ✦** é uma aplicação leve que funciona direto no navegador mobile, sem login ou instalação. Ideal para acompanhar seus gastos, entradas e entender seus hábitos financeiros no dia a dia.

Focado em:
- praticidade no celular 📱
- visual clean e agradável 🌸
- uso rápido no dia a dia

---

## 📊 Funcionalidades

### ➕ Registro de lançamentos
- Adicione **gastos** ou **entradas**
- Descrição, valor, data e categoria
- Interface rápida e otimizada para toque

### 📈 Dashboard financeiro
- Resumo por período (7 dias, mês ou total)
- Saldo atualizado automaticamente
- Indicador de orçamento utilizado 📊
- Gráfico de gastos por categoria 🍰
- Gastos por dia da semana

### 📋 Transações
- Lista completa de lançamentos
- Filtro por tipo (gastos ou entradas)
- Exclusão de registros

### 💾 Armazenamento local
- Dados salvos no navegador
- Sem login, sem backend

---

## 📱 Como usar (mobile only)

Para melhor experiência, use como um app:

1. Abra o arquivo no navegador do celular (Chrome ou Safari)
2. Toque no menu (⋮ ou compartilhar)
3. Selecione **"Adicionar à tela inicial"**
4. Abra direto pela home do celular ✨

Isso transforma o planner em um **pseudo-app**, com acesso rápido e experiência mais fluida 💖

---

## ⚠️ Importante

- 🚫 Não é otimizado para desktop
- 📱 Uso recomendado apenas no celular
- 🔒 Dados ficam apenas no dispositivo
- 🧹 Limpar o navegador pode apagar os dados
